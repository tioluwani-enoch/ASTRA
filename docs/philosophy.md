# Philosophy

Astra is designed with a few core principles:

---

## 1. Usefulness First

Every feature must provide real, daily value.

---

## 2. Memory Over Interface

A smart system matters more than a polished UI.

---

## 3. Controlled Proactivity

Astra should assist, not interrupt unnecessarily.

---

## 4. Iterative Growth

Start small, improve continuously.

---

## 5. Human-in-the-Loop

The user always has final control over decisions.
