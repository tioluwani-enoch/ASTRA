# Astra Overview

Astra is a personal AI assistant designed to function as a **persistent, intelligent system** that supports daily decision-making and task execution.

Unlike traditional assistants, Astra emphasizes:

* Long-term memory
* Context awareness
* Structured planning
* Controlled autonomy

---

## System Identity

Astra acts as:

* A **planner** (organizes your day)
* A **tracker** (remembers tasks and goals)
* A **reasoner** (helps make decisions)
* An **executor** (performs structured actions)

---

## Core Loop

Astra operates on a continuous loop:

1. Plan → Generate daily structure
2. Act → Execute and track tasks
3. Reflect → Learn from outcomes

This loop improves performance over time.
