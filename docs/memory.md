# Memory System

Memory is a foundational component of Astra.

---

## Memory Types

### 1. Permanent

* Preferences
* Long-term goals

### 2. Semi-Permanent

* Active projects
* Ongoing tasks

### 3. Disposable

* Temporary notes
* Session data

---

## Goals

* Maintain continuity across sessions
* Improve decision-making over time
* Reduce repetitive input from user
