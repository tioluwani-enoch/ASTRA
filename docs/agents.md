# Agents

Agents are specialized reasoning modules within Astra.

---

## Planner Agent

* Breaks goals into actionable steps
* Generates schedules

---

## Reflection Agent

* Analyzes completed vs planned tasks
* Suggests improvements

---

## (Future) Executor Agent

* Performs actions on behalf of the user

---

## Design Philosophy

Agents should be:

* Focused
* Modular
* Replaceable
